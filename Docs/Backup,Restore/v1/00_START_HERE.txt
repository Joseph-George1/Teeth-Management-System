## 🚀 QUICK START GUIDE

### PRODUCTION SERVER - BACKUP (5 minutes)
```bash
cd /path/to/teeth-management/backup
export DB_PASSWORD="your_oracle_sys_password"
chmod +x backup.sh
./backup.sh
tail -f /backup/logs/backup_*.log        # Monitor progress
```

### NEW SERVER - RESTORE (5 minutes)
```bash
cd /path/to/backup
sudo bash setup.sh restore-server        # Install dependencies
export DB_PASSWORD="your_oracle_sys_password"
chmod +x restore.sh
./restore.sh /path/to/backup
tail -f /var/log/teeth-management/restore/restore_*.log  # Monitor
```

### SYSTEM SETUP (30 minutes)
```bash
sudo bash setup.sh                       # Install all dependencies
sudo systemctl start apache2
sudo systemctl start tomcat
sudo systemctl start oracle-xe-21c
```
