# Migration Quick Reference - Oracle XE

## 📋 One-Line Commands

### Backup Database
```bash
rman target / << EOF
BACKUP DATABASE PLUS ARCHIVELOG;
EXIT;
EOF
```

### Run Migration Script
```bash
sqlplus system/your_password@XE @/path/to/migration_oracle_xe.sql
```

### Verify Migration
```bash
sqlplus system/your_password@XE
DESC appointments;
DESC requests;
SELECT index_name FROM user_indexes WHERE table_name = 'APPOINTMENTS';
EXIT;
```

## 🎯 What Gets Added To appointments Table

| Column | Type | Purpose |
|--------|------|---------|
| request_id | NUMBER | Link to request (what service) |
| created_at | TIMESTAMP | When booked (for 7-day expiration) |
| is_expired | NUMBER(1) | 1=auto-cancelled, 0=active |
| is_history | NUMBER(1) | 1=done/cancelled, 0=active |

Duration & Notes → Set to NULL (no longer tracked)

## ⚠️ If Something Goes Wrong

**Script fails?**
- It won't break - non-existent columns/indexes are skipped
- Run DESC appointments to check current state
- Restore from backup if major issues

**Columns don't appear?**
- Exit and reconnect: `EXIT;` → `sqlplus system/password@XE`
- Run: `COMMIT;`
- Check again: `DESC appointments;`

**Need to rollback?**
- Drop new columns: `ALTER TABLE DROP COLUMN is_expired;`
- Drop constraint: `ALTER TABLE DROP CONSTRAINT fk_appointment_request;`
- Restore from backup: RMAN recovery
